# Instagram User Behavior and Influencer Mapping (SQL)

##  Overview
Performed SQL-based analysis on 1,500+ interactions across 80 users to derive engagement insights, influencer segmentation, and content performance patterns.

##  Key Analysis
- Computed Engagement Score using SQL aggregations  
- Segmented users into 4 tiers (Top / Strong / Growing / Emerging)  
- Identified high-impact influencers and content themes  

##  Insights
- Identified 6 high-impact influencers  
- Key content themes: #smile, #party, #beach, #adopted  
- Enabled targeted engagement strategies  

##  Tools Used
- SQL (Joins, GROUP BY, CASE, Aggregations)
